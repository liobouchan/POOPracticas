Práctica 13 Intrumentación de Hilos

Objetivo: El alumno desarrollara programación orientada a objetos que le permita utilizar hilos, con lo que le permitirá observar las características de los objetos Thread.

Introducción Teórica

Al igual que casi todos los elementos que componen un applet, un hilo es un objeto que corresponda a una definición de clase.

Las instrucciones que ejecuta un applet para realizar una tarea específica definen el hilo de ejecución del applet. Un applet con varios hilos es un applet que corre dos hilos o más, al mismo tiempo. Para crear un hilo de ejecución es necesario declarar un objeto Thread (Hilo), pasando a la función constructora de la clase Thread un objeto cuyas instrucciones quiere usted que ejecute el hilo.

Para ejecutar un objeto Thread los applets llaman a la función start (iniciar) de la clase Thread. Esta función, a su vez, llama a la función run (correr o ejecutar) del objeto.

Un Thread se declara en el applet declarando un objeto Thread, como se muestra enseguida:

Thread UnHilo = null;

En este caso, la declaración crea un objeto Thread llamado UnHilo. Después, la instrucción inicializa el objeto en null para  indicar al código que sigue que el applet aún no ha creado un objeto-instancia de la clase Thread mediante el operador new. Al asignar el valor null a una variable-objeto, el código puede revisar dentro de las funciones start y stop si existe el hilo. De ser así, la variable contendrá el valor null.
